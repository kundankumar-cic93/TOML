# Streamlit Ollama Chat Application

## Overview
The Streamlit Ollama Chat Application is a powerful tool that allows users to interact with AI models in real-time. Built with Streamlit, this application provides an intuitive interface for chat-based interactions.

## Features
- **User-friendly Interface:** Simple UI for seamless interaction.
- **Real-time Responses:** Get instant replies from the integrated AI models.
- **Model Selection:** Choose from various pre-trained models based on user requirements.

## Installation
You can easily set up the Streamlit Ollama Chat Application by following these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/kundankumar-cic93/TOML.git
   cd TOML
   ```
2. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the application:
   ```bash
   streamlit run app.py
   ```

## Usage
1. Launch the application by navigating to the appropriate URL.
2. Select your preferred model.
3. Start typing your queries in the chat interface.
4. Receive responses in real-time from the selected AI model.
